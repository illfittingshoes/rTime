Namespace: get

get

Holds function properties for extracting individual values from a given date in UTC time. All return values are retrieved using Date.prototype.getUTC[field](). Has an alias called "getUTC"
Source:
  • rTime.js, line 239
See:
  • rTime~getUTC

Example

// get the year of a given date
rTime.get.year(someDate)
